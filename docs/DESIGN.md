# Love Letter - 디자인 문서

> UI/UX 설계 및 화면 구성

## 1. 디자인 원칙

### 1.1 핵심 가치
- **따뜻함**: 부드러운 색상, 둥근 모서리
- **심플함**: 핵심 기능에 집중, 불필요한 요소 제거
- **감성적**: 편지를 주고받는 설렘과 기대감 표현
- **접근성**: 누구나 쉽게 사용 가능

### 1.2 디자인 시스템

#### 색상 팔레트
```
Primary:     #FF6B9D (로즈 핑크)
Secondary:   #FFB5C5 (연한 핑크)
Background:  #FFF9FA (크림 화이트)
Text:        #4A4A4A (다크 그레이)
Accent:      #FF4081 (핫 핑크)
Success:     #4CAF50 (그린)
Error:       #F44336 (레드)
```

#### 타이포그래피
```
Heading:  Pretendard Bold, 24-32px
Body:     Pretendard Regular, 14-16px
Caption:  Pretendard Light, 12px
Letter:   나눔명조 또는 손글씨체 (편지 본문)
```

#### 스페이싱
```
xs: 4px
sm: 8px
md: 16px
lg: 24px
xl: 32px
```

## 4. 컴포넌트 라이브러리

### 4.1 공통 컴포넌트

| 컴포넌트 | 설명 | 변형 |
|---------|------|------|
| Button | 기본 버튼 | primary, secondary, ghost |
| Input | 입력 필드 | text, password, textarea |
| Card | 카드 컨테이너 | default, elevated |
| Badge | 뱃지/라벨 | new, read, count |
| Avatar | 프로필 이미지 | small, medium, large |
| Modal | 모달 다이얼로그 | alert, confirm, form |
| Toast | 알림 토스트 | success, error, info |
| Tabs | 탭 네비게이션 | default |
| BottomNav | 하단 네비게이션 | default |

### 4.2 도메인 컴포넌트

| 컴포넌트 | 설명 |
|---------|------|
| LetterCard | 편지 목록 아이템 |
| LetterEditor | 편지 작성 에디터 |
| LetterViewer | 편지 읽기 뷰어 |
| EnvelopeAnimation | 봉투 열기 애니메이션 |
| ThemePicker | 편지지 선택기 |
| MediaUploader | 미디어 업로드 |
| AnniversaryCard | 기념일 카드 |
| CoupleConnector | 커플 연결 UI |

## 5. 인터랙션 & 애니메이션

### 5.1 핵심 인터랙션

| 동작 | 애니메이션 |
|------|----------|
| 봉투 열기 | 봉투가 열리면서 편지가 나옴 (0.5s) |
| 편지 전송 | 편지가 날아가는 효과 (0.3s) |
| 새 편지 도착 | 바운스 + 글로우 효과 |
| 페이지 전환 | 슬라이드 (0.2s) |
| 버튼 탭 | 스케일 0.95 + 배경색 변화 |

### 5.2 마이크로인터랙션

- 하트 아이콘 탭: 펄스 효과
- 스크롤: 스무스 스크롤
- 로딩: 편지 아이콘 회전
- 성공: 체크마크 + 컨페티

## 6. 반응형 디자인

### 6.1 브레이크포인트
```
Mobile:  320px - 767px (기본)
Tablet:  768px - 1023px
Desktop: 1024px+
```

### 6.2 레이아웃 변화

| 화면 | Mobile | Desktop |
|------|--------|---------|
| 편지 목록 | 1열 | 2-3열 그리드 |
| 편지 읽기 | 풀스크린 | 가운데 정렬 (max-width: 600px) |
| 보관함 | 리스트 | 그리드 |
| 네비게이션 | 하단 탭 | 사이드바 |

## 7. 접근성

- 색상 대비: WCAG AA 기준 충족
- 포커스 표시: 명확한 포커스 링
- 터치 영역: 최소 44x44px
- 스크린 리더: 적절한 aria-label
- 키보드 네비게이션 지원
